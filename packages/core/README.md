# @stencil-pm/core

The portable, zero-tool-dependency TypeScript library for Stencil template management.

## Overview

This package provides the core business logic for parsing, validating, and resolving Stencil templates. It has no dependency on any specific AI tool or editor — adapters (VS Code, Claude Code, Codex) build on top of this package.

## Status

Under active development. Public API is not yet stable.
